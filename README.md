To run this project, you need Premake and Conan, which have been provided in the class slides.


To compile use ./tools/genprojectwin64.bat


To run the proyect, open ./build/MotArda.sln and press ctrl + F5.


./example folder contains different examples of how to implements all features of the engine.

The MD file with the structure and information of the engine is at ./doc/MANUAL.md

You can check ./examples/testScripting.cpp to learn how to bind components in order to 
add Scripting to your code.
